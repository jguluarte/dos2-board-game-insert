# Card Box Groupings — cut sheet for standard (63x88) cards

Mirror of `location-box-groupings.md`, for the 63x88 standard-card wells.
Each box = a flat list of inner sections (one row = one divider section).
Header shows section count at a glance; **Total** = cards in the box.

All counts are **physical cards in sleeves** (summed `Quantity` from
`~/Documents/DOS Rules/*.md`).
